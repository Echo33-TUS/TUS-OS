TUS OS

TUS OS is a protocol and reference implementation for producing verifiable decision receipts: canonical, content-addressable records of a decision step that can be independently inspected and validated.

A receipt captures, at minimum:

the evaluated state,

the permitted local moves,

the scoring and tie-break rules applied, and

the exact transition selected (with ranked comparisons where applicable).

TUS OS is designed to support auditable AI-assisted analysis. Receipts act as a binding substrate for downstream interpretation: they constrain and structure how an AI (or any renderer) may evaluate a situation, so that narrative outputs remain traceable, repeatable, and reviewable rather than free-form.

This repository currently publishes the public concept note and integrity artifacts (PDF + SHA-256). Normative specifications, conformance materials, and open-source releases will be published separately.
